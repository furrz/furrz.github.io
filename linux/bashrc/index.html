nova's bash prompt

My bash prompt is very minimalist: 

PROMPT_COLOR_ROOT="\033[31m"
PROMPT_COLOR_USER="\033[34m"

if [[ "$EUID" -ne 0 ]]; then
	PROMPT_COLOR=$PROMPT_COLOR_USER
else
	PROMPT_COLOR=$PROMPT_COLOR_ROOT
fi

PS1="\n $PROMPT_COLORλ\033[0m "
This instruction creates a single lambda symbol λ. It is coloured blue for a regular user, and red for root.
